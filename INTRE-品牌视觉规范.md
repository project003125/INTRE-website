# INTRE品牌视觉规范

## 摘要

本文档是 INTRE 项目的品牌视觉权威规范（D1a）。它定义了品牌标识系统的设计理念、视觉要素、使用规范与禁止事项，确保 INTRE 在所有触点上的一致性表达。品牌设计遵循**包豪斯内核 + 苹果简约**的双重美学原则：包豪斯提供几何构成与功能决定形式的经典内核，苹果提供极致减法与呼吸感的潮流表达。二者冲突时，以包豪斯为准。

**关键词**：品牌规范；视觉标识；Logo；色彩系统；排版系统；包豪斯；极简主义

---

## 一、品牌理念

### 1.1 品牌定位

INTRE（Integrated Psychological Layer Engineering）是功能层心理学（FLP）的工程实现框架。它的核心贡献是在神经科学与心理学之间建立了一个全新的抽象层级——**功能层**，为心理现象提供形式化、可计算、可累积的基础语言。

INTRE 的品牌人格特征：

| 维度 | 描述 |
|------|------|
| 智力风格 | 架构式、形式化、系统性 |
| 交流方式 | 精确、克制、无废话 |
| 视觉气质 | 几何理性、呼吸感、学术权威 |
| 情感基调 | 冷静、自信、不迎合 |

### 1.2 设计哲学

INTRE 品牌设计遵循双重美学原则：

**包豪斯内核**（优先级：高）
- 功能决定形式：每个视觉元素必须承载理论意义
- 几何构成：以基本几何形（三角、六边形、圆）为构件
- 少即是多：删除一切不服务于信息的装饰
- 类型化：建立可复用的视觉语法而非个案设计

**苹果简约**（优先级：次高）
- 极致减法：能用一条线表达的绝不用两条
- 呼吸感：大量留白，内容密度适中
- 精致细节：0.5px 线宽、精确的间距节奏
- 触觉质感：微妙的交互反馈、克制的阴影

**冲突裁决**：当二者冲突时（如包豪斯要求功能语义丰富度 vs 苹果要求极致减法），以包豪斯为准——形式必须先服务于意义的传达，然后才追求简约。

### 1.3 品牌叙事

INTRE 的品牌叙事围绕一个核心隐喻展开：**功能层是棱镜**。

正如白光穿过棱镜折射出光谱，心理现象通过功能层的分析折射出可计算的结构。这个隐喻贯穿 Logo 设计、色彩系统与排版节奏。

## 二、品牌标识（Logo）

### 2.1 设计概念

INTRE Logo 采用**「功能层棱镜」**（Functional Prism）方案，经过多轮方案比选后确定。Logo 由三层视觉语义叠加构成：

| 层级 | 几何形 | 理论对应 | 视觉权重 |
|------|--------|----------|----------|
| 外轮廓 | 等边三角形 | 三层架构（现象层-功能层-神经层） | 最重（3.2px 描边） |
| 内结构 | 正六边形 + 6 节点 | M1-M6 六个功能模块 | 中等（1.6px 描边，0.6 透明度） |
| 核心 | 实心圆点 | Ψ(t) 功能状态向量 | 实心填充 |
| 氛围弧 | 三段淡弧线 | N-S-R 元公理循环暗示 | 最轻（1px，0.3-0.4 透明度） |

**设计决策记录**：经过四套方案比选（棱镜/层叠/回路/完整标识），最终选定方案1「棱镜」作为主符号，方案4「完整标识」作为包含字标的组合版本。方案2（层叠）和方案3（回路）的视觉语义对普通受众过于抽象，未采用。

### 2.2 标准版

**文件**：`assets/logo.svg`（200×200，单色 Navy）

使用场景：
- 网站导航栏
- 文档页眉
- 学术海报
- 演示文稿封面

### 2.3 深色版

**文件**：`assets/logo-dark.svg`（200×200，白色线条）

使用场景：
- Navy 深色背景
- 深色模式界面
- 视频水印

### 2.4 完整标识

**文件**：`assets/logo-wordmark.svg`（720×200，Logo + 字标）

组合规范：
- Logo 图标与「INTRE」字标水平排列
- Logo 图标垂直中心对齐字标视觉中心
- Logo 与字标间距：约 Logo 宽度的 30%

### 2.5 Favicon

**文件**：`assets/favicon.svg`（32×32，简化版）

简化规则：仅保留三角轮廓 + 六边形 + 核心点，移除 NSR 弧和模块节点，确保 16px 尺度可识别。

### 2.6 最小尺寸

| 版本 | 最小尺寸 | 说明 |
|------|----------|------|
| Logo 标准版 | 32×32px | 低于此尺寸节点不可辨 |
| Logo + Wordmark | 宽 360px | 字标低于此宽度不可读 |
| Favicon | 16×16px | 已为小尺寸简化 |

### 2.7 安全区域

Logo 四周需保留不小于 Logo 高度 25% 的留白空间。在该区域内不得放置任何文字、图形或页面边缘。

### 2.8 禁止事项

| 编号 | 禁止操作 | 原因 |
|------|----------|------|
| P-01 | 改变 Logo 长宽比 | 破坏几何比例与理论映射 |
| P-02 | 添加投影、发光、渐变 | 违反包豪斯平面原则 |
| P-03 | 旋转 Logo | 三角形朝向承载理论语义 |
| P-04 | 在 Logo 上叠加文字 | 破坏可读性 |
| P-05 | 使用非规范色彩 | 破坏品牌一致性 |
| P-06 | 将 Logo 置于低对比度背景 | 破坏可识别性 |
| P-07 | 截取 Logo 部分使用 | 每个元素承载独立语义，不可分离 |

## 三、色彩系统

### 3.1 品牌主色

INTRE 品牌色为 **Navy** 色系，取自心理科学与学术权威的传统色彩联想，同时避免与任何现有心理学流派标识色冲突。

| 色名 | 色值 | 用途 |
|------|------|------|
| Navy | `#1F4880` | 主色：Logo、标题、关键交互、导航 |
| Navy Light | `#2A5DA0` | 主色悬浮态 |
| Blue | `#3B6FB5` | 辅助色：次要信息、NSR 弧 S 段 |
| Blue Light | `#6DA3D4` | 辅助色浅版：NSR 弧 R 段、装饰线 |

### 3.2 中性色

| 色名 | 色值 | 用途 |
|------|------|------|
| Slate 900 | `#0f172a` | 正文主色 |
| Slate 700 | `#334155` | 二级正文 |
| Slate 500 | `#64748b` | 辅助文字、说明 |
| Slate 400 | `#94a3b8` | 占位符、禁用态 |
| Slate 200 | `#e2e8f0` | 分割线 |
| Slate 100 | `#f1f5f9` | 背景色 |
| White | `#ffffff` | 页面背景、深色背景文字 |

### 3.3 通道色

四个核心模块的通道色彩，用于徽章、标签与图表：

| 模块 | 通道色 | 色值 | 语义 |
|------|--------|------|------|
| UPLS | 语义通道绿 | `#10b981` | 知识与语言 |
| UNIS | 神经通道蓝 | `#3b82f6` | 神经信号 |
| UBMS | 行为通道橙 | `#f59e0b` | 可观测行为 |
| PSE | 计算引擎紫 | `#8b5cf6` | 仿真与计算 |
| REVER | 伦理约束琥珀 | `#d97706` | 规范与约束 |

### 3.4 色彩使用原则

1. **单色优先**：Logo、标题、核心交互仅使用 Navy 色系
2. **通道色节制**：仅在标识模块身份时使用通道色，不得作为大面积填充
3. **渐变禁止**：所有色彩使用纯色填充，不使用渐变（包豪斯原则）
4. **对比度底线**：文字与背景对比度 ≥ 4.5:1（WCAG AA）

## 四、排版系统

### 4.1 字体栈

| 用途 | 字体 | 回退栈 |
|------|------|--------|
| 主字体 | Inter | `-apple-system, BlinkMacSystemFont, sans-serif` |
| 等宽字体 | JetBrains Mono | `Consolas, Monaco, monospace` |

**选型理由**：
- Inter：x-height 高、字距适中、OpenType 特性完善，覆盖学术与工程双重语境
- JetBrains Mono：等宽字体中可读性最佳，适合公式、代码、状态向量展示

### 4.2 字号层级

| 级别 | 用途 | 尺寸 | 字重 | 行高 | 字距 |
|------|------|------|------|------|------|
| H1 | 页面标题 | 2.5-3.2rem | 700 | 1.1 | -0.03em |
| H2 | 章节标题 | 1.5-1.8rem | 600 | 1.3 | -0.02em |
| H3 | 小节标题 | 1.15-1.25rem | 600 | 1.4 | -0.01em |
| Body | 正文 | 1rem | 400 | 1.7 | 0 |
| Caption | 说明文字 | 0.75-0.875rem | 400-500 | 1.5 | 0-0.08em |
| Mono | 公式/代码 | 0.875rem | 400 | 1.6 | 0 |

### 4.3 排版原则

1. **标题紧缩**：H1-H2 使用负字距，形成视觉张力
2. **正文呼吸**：行高 1.7 确保长文可读性
3. **层级分明**：字重跳跃（700→400）而非渐变（400→500→600）
4. **数字对齐**：表格数据使用 JetBrains Mono 等宽对齐

## 五、图形系统

### 5.1 几何语汇

INTRE 的视觉语言由三个基本几何形构成：

| 几何形 | 语义 | 使用场景 |
|--------|------|----------|
| 三角形 | 三层架构、层级关系 | Logo、架构图、分隔符 |
| 六边形 | 功能模块、系统结构 | 模块图标、流程图节点 |
| 圆 | 状态向量、核心概念 | 核心标记、数据点、强调符 |

### 5.2 线宽体系

| 级别 | 线宽 | 用途 |
|------|------|------|
| L1 | 2.5-3.5px | Logo 轮廓、核心架构线 |
| L2 | 1.5-2px | 模块边框、辅助结构线 |
| L3 | 0.75-1px | 连接线、装饰线、网格 |
| L4 | 0.5px | 卡片边框、分割线 |

**原则**：线宽跳跃（如 2px→0.5px）而非渐变（1px→1.5px→2px），遵循包豪斯类型化原则。

### 5.3 圆角体系

| 级别 | 圆角 | 用途 |
|------|------|------|
| R1 | 0 | Logo 几何形（不圆角） |
| R2 | 6px | 按钮、标签、小型交互元素 |
| R3 | 8-12px | 卡片、面板、模块容器 |
| R4 | 16-20px | 全宽横幅、大区块 |

## 六、布局系统

### 6.1 网格

INTRE 采用 12 列网格系统，基于 4px 基础单位：

| 断点 | 容器宽度 | 列宽 | 槽宽 |
|------|----------|------|------|
| 移动端 <768px | 100% | 流式 | 16px |
| 平板 768-1199px | 720px | 60px | 16px |
| 桌面 ≥1200px | 960-1180px | 72px | 20px |

### 6.2 间距节奏

基于 4px 基数，使用 2 倍递增：

| 级别 | 间距 | 用途 |
|------|------|------|
| S1 | 4px | 图标与文字间距 |
| S2 | 8px | 行内元素间距 |
| S3 | 12px | 组内元素间距 |
| S4 | 16px | 区块内边距 |
| S5 | 24px | 区块间距 |
| S6 | 32px | 章节间距 |
| S7 | 48px | 页面级间距 |
| S8 | 64px | 英雄区间距 |

### 6.3 留白原则

1. **功能留白**：留白服务于信息层级，不是装饰
2. **节奏留白**：间距遵循 2^n 递增，形成视觉节奏
3. **呼吸留白**：关键信息周围留白面积 ≥ 元素自身面积 50%

## 七、交互规范

### 7.1 动效原则

| 原则 | 实现 |
|------|------|
| 克制 | 仅在功能性需要时添加动效 |
| 快速 | 过渡时长 150-300ms |
| 自然 | 使用 cubic-bezier(0.16, 1, 0.3, 1) 缓动 |
| 可逆 | 所有状态变化可回退 |

### 7.2 微交互

| 元素 | 触发 | 效果 | 时长 |
|------|------|------|------|
| 导航链接 | hover | 背景色 rgba(255,255,255,0.08) | 150ms |
| 卡片 | hover | Y偏移 -2px + 阴影增强 | 300ms |
| 按钮 | hover | 背景色加深 | 150ms |
| 侧边栏目录 | scroll | 当前章节高亮 | 即时 |

### 7.3 无障碍

1. 所有交互元素支持键盘操作
2. 焦点指示器：2px 实线 Navy 轮廓 + 2px 偏移
3. 动效尊重 `prefers-reduced-motion`
4. 文字与背景对比度 ≥ 4.5:1（WCAG AA）

## 八、应用场景

### 8.1 网站导航栏

- Logo（深色版）置于左侧

- Logo + Wordmark 置于左上角
- 最小高度 60px

### 8.3 文档模板

- 页眉 Logo 高度 24px
- 位置：左对齐
- 颜色：Navy `#1F4880`

### 8.4 社交媒体

- 头像：Logo 标准版，正方形裁切
- 封面：Logo + Wordmark 居中
- 通道色可用于封面装饰元素
