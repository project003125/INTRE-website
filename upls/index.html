UPLS

统一心理语言系统 · 形式化元语言基础设施 语义通道
UPLS 提供一套严格的形式化语法，使心理状态可描述、可计算、可干预。它包含内感架构 M‑R‑D‑S、操作架构 G‑M‑A‑R 以及知识基础设施 PCUI/PSA/PSL，是 INTRE 框架的元语言基础。

一、M‑R‑D‑S 内感架构

描述心理状态内部组成的语法：M（六大功能模块）、R（资源层）、D（调度层）、S（结构层）。

1.1 六大功能模块（M层）

模块英文中文功能定位系统角色
M1Embodied具身物理基座与能量引擎提供生理能量基底，约束其他模块
M2Cognitive认知建模引擎与知识库存储信念、知识，进行推理
M3Volition意志中央调度器目标设定、优先级分配
M4Emotional情绪价值评估中心对内外事件赋予效价
M5Linguistic言语符号接口与社会连接语言生成与理解，社会沟通
M6Behavioral行为执行系统动作输出与习惯执行

1.2 资源层（R层）

组件英文中文时间特征功能定义
RES-DDispatchable Resource调度资源小时-天（快速波动）即时可用的心理燃料（注意力、意志力）
RES-BBasal Resource基底资源周-月（缓慢变化）长期心理资本（自尊、希望、韧性）

1.3 调度层（D层）

组件英文中文运行特征功能定义
DSP-FForeground Dispatch前台调度需意识参与目标导向的主动控制进程
DSP-BBackground Dispatch后台调度自动运行本能反应与习惯化进程

1.4 结构层（S层）

组件英文中文系统角色功能定义
STR-RRules Structure规则结构系统规则内容指导决策与评估的逻辑算法（如“我必须完美”）
STR-CConnections Structure连接结构系统关联网络存储元素间的关系权重（如“失败”与“无价值”的强连接）
STR-SSequence Structure序列结构系统执行流程定义行为或反应的序列步骤（如习惯链）

  M3: { DSP-F: 0.3, DSP-B: 0.8 },

}

二、G‑M‑A‑R 操作架构

原子级心理操作语法，对应系统动力学的四种基本干预类型。

GATE · 门控

作用对象： DSP、STR-S

操作本质： 通道开关控制（启动/抑制进程）

示例： GATE(DSP-B, -) —— 抑制后台自动思维

MODULATE · 调制

作用对象： RES

操作本质： 资源数量调节（增加/减少）

示例： MODULATE(RES-D, +0.3) —— 提升调度资源

ALIGN · 对齐

作用对象： STR-R

操作本质： 规则修正（更新信念）

示例： ALIGN(STR-R, "我不必完美")

RELINK · 重联

作用对象： STR-C

操作本质： 连接重构（改变关系权重）

示例： RELINK(STR-C, "失败"→"学习", +0.6)

正交性证明： G-M-A-R 四操作分别作用于 进程（DSP/STR-S）、存量（RES）、规则（STR-R）、连接（STR-C），四个维度互不重叠，覆盖所有心理干预的基本形式。

三、知识基础设施

3.1 PCUI · 心理概念唯一标识符

为每个心理学概念分配唯一编码，消除歧义，支持跨流派映射。

PCUI概念名称所属流派对应 M-R-D-S 位置
PCUI-0012自动化思维CBTM2-STR-R（规则结构）
PCUI-0037情绪效价情绪心理学M4（情绪模块）
PCUI-0089自我效能感社会认知理论M3-RES-B（基底资源）

3.2 PSA · 心理语义网络

定义概念间的关系（因果、关联、层级），形成可计算的知识图谱。

// PSA 关系示例
PCUI-0012 (自动化思维) --[causes]→ PCUI-0037 (负性情绪)
PCUI-0089 (自我效能感) --[modulates]→ M3-DSP-F (前台调度)

3.3 PSL · 心理专项词汇库

自然语言与形式化 PCUI 的映射词典，支持临床文本的自动编译。

自然语言表达映射 PCUI上下文约束
"我感到筋疲力尽"PCUI-0056 (能量耗竭) → M1-RES-D↓资源层下降
"我什么都做不好"PCUI-0012 (自动化思维) → M2-STR-R固化规则层负性信念

四、跨流派整合

心理学流派核心概念UPLS 映射
CBT（认知行为疗法）自动化思维、核心信念M2-STR-R（规则结构）
DBT（辩证行为疗法）情绪调节、痛苦耐受M4（情绪）+ M3-DSP-F（前台调度）

UPLS 的整合价值： 不同流派的概念不再是不可通约的“语言游戏”，而是可以映射到同一套形式化语法（M-R-D-S、G-M-A-R）的不同参数配置。这为心理学知识的累积和跨流派对话提供了工程基础。
