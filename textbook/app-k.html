附录K：PSA依赖——矩阵、级联模板与文化参数

定位：第15章（语义架构）的配套速查手册，提供可直接用于仿真、教学和临床推理的标准依赖矩阵与级联模板。

学习目标：能够查阅六模块标准依赖矩阵进行级联推理；能够识别一个症状属于哪类级联模板；能够判断操作-组件映射是否合法。

K.2 四类级联路径模板

级联路径模板描述失调从起始模块到下游模块的典型传播模式。每类模板包含：识别口诀、级联方向、典型症状指纹、最高杠杆靶点、误判风险。

模板名称 口诀 级联方向 典型症状指纹 最高杠杆干预靶点 误判风险
资源崩溃型 “先累后拖再冲动” M1-RES-D↓ → M3-RES-D↓ → M3-DSP-F失效 → M6-DSP-B↑ 拖延、冲动消费、决策疲劳 MODULATE+ M1-RES-D（睡眠/营养） 易与情绪驱动型混淆，区别在M4-STR-C是否激活
情绪驱动型 “先怕后累再反刍” M4-STR-C激活 → M4-DSP-B↑ → M3-RES-D↓ → M2-DSP-B↑ 焦虑、反刍、回避行为 RELINK M4-STR-C 或 GATE– M4-DSP-B 易与资源崩溃型混淆，区别在级联起点（M4 vs M1）
习惯固化型 “先做后忘再失控” S-P触发 → M6-DSP-B重复 → M6-STR-S巩固 → M3-DSP-F门控失效 成瘾、强迫行为、习惯性拖延 GATE– S-P触发 + GATE+替代行为 易与资源崩溃型混淆，区别在M1资源是否正常
规则扭曲型 “先信后怕再逃避” M2-STR-R错误规则 → M4-STR-R威胁评估扭曲 → M4-DSP-B↑ → 回避行为 完美主义、灾难化、社交焦虑 ALIGN M2-STR-R（认知重构） 易与情绪驱动型混淆，区别在M2-STR-R是否为根源

K.3 反向溯源速查表

给定末端症状，反向匹配最可能的级联模板（用于临床快速定位）。

末端症状 最可能模板（按概率降序） 关键区分指标 快速验证方法
拖延 资源崩溃型 > 情绪驱动型 > 规则扭曲型 M3-RES-D vs M4-STR-C vs M2-STR-R 问“是累、怕、还是觉得必须完美？”
反刍 情绪驱动型 > 资源崩溃型 M4-DSP-B vs M3-RES-D 问“是焦虑触发还是疲劳后失控？”
冲动消费/进食 习惯固化型 > 资源崩溃型 M6-STR-S vs M3-RES-D 问“是习惯动作还是自控力耗尽？”
回避社交 情绪驱动型 > 规则扭曲型 M4-STR-C vs M2-STR-R 问“是怕被伤害还是觉得自己不行？”
失眠 资源崩溃型 > 情绪驱动型 M1-RES-D节律 vs M4-DSP-B 问“是身体不累还是脑子停不下来？”
完美主义 规则扭曲型 M2-STR-R 问“是否认为不完美就是失败？”
成瘾复发 习惯固化型 > 资源崩溃型 S-P触发 vs M3-RES-D 问“是看到线索就自动做，还是意志薄弱时做？”
惊恐发作（急性） 情绪驱动型 M4-DSP-B 问“是否有突然的恐惧反应，无前兆？”

溯源决策树： 1. 是否存在明显的环境触发（S-P）？是 → 优先考虑习惯固化型或情绪驱动型 2. 资源水平（M1/M3-RES-D）是否低于30？是 → 优先考虑资源崩溃型 3. 是否存在稳定的错误信念（M2-STR-R）？是 → 优先考虑规则扭曲型 4. 是否存在创伤性连接（M4-STR-C高权重）？是 → 优先考虑情绪驱动型

K.4 操作-组件映射合法性矩阵

本矩阵规定G-M-A-R四类操作的合法靶点、常见误用及替代方案。违反合法性将导致干预无效或医源性损害。

操作 直接合法靶点 常见误用 误用后果 替代操作
GATE DSP-F, DSP-B（运行中的过程）；运行中的STR-S序列 GATE STR-R（“门控一个信念”） 无效，信念不是进程 ALIGN
同上 GATE STR-C（“门控一个连接”） 无效，连接不是进程 RELINK
同上 GATE未被触发的STR-S（存储态序列） 无效，序列未运行 反复GATE–执行触发过程来削弱
MODULATE RES-D（所有模块） MODULATE RES-B 无效，基底资源不能直接补充 长期RELINK/ALIGN + 环境输入
同上 MODULATE STR 无效，结构不能通过资源补充直接改变 ALIGN/RELINK
ALIGN STR-R（所有模块） ALIGN STR-C “道理都懂但感觉不变” RELINK
同上 ALIGN STR-S 无法直接改写程序性知识 GATE+重复 + 行为排练
RELINK STR-C（所有模块） RELINK STR-R “体验了但无法概括” ALIGN
同上 RELINK STR-S 无法直接改写序列 GATE+重复 + 消退

临床快速判断： - 问题表现为“我做不到”且资源充足 → 查GATE/RELINK - 问题表现为“我不信”且资源充足 → 查ALIGN - 问题表现为“我太累” → 先查MODULATE，勿跳级

K.5 PSL-L文化变体参数速查表

文化变体参数在PSL-L和PCUI层注入，用于调整MCT编译和PSE生态位先验。以下参数为F级（跨文化心理学文献推导），有待PSE-C多文化样本校准（P级）。

参数 东亚 南亚 拉美 西欧/北美（基线） 说明
焦虑躯体化权重（M1-DSP-B） ×1.30 ×1.15 ×1.10 ×1.00 焦虑更多表现为躯体不适
情绪表达权重（M4-DSP-F） ×0.85 ×0.90 ×1.10 ×1.00 内敛 vs 外放的文化规范
依恋安全感基线（M4-RES-B均值偏移） -5 -3 +2 0 集体主义中安全感更多来自关系
自我叙事独立性（M5-STR-R） 互依型 混合型 独立型 独立型 自我构念差异
求助行为权重（M5-DSP-F） ×0.70 ×0.80 ×0.90 ×1.00 文化对心理求助的污名化程度
社会赞许性偏差（观测噪声） 高 中高 中 中低 自陈量表中的偏差程度

使用说明： - 文化变体参数在PSE-I阶段注入，影响生态位匹配和MCT编译 - 当个体文化背景与生态位库默认文化不匹配时，系统自动应用相应调整 - 若个体具有多重文化身份，需使用加权组合或单独校准 - 所有参数为初始先验，实际应用时应随个体数据逐步校准后移

文化变体使用的伦理警示：文化参数不得用于刻板印象推断。仅当个体明确认同该文化背景时才应用；当信息不足时，使用更宽的先验分布而非强制应用文化调整。
