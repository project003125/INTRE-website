UBMS

统一行为测量系统 · 结构化行为观测到功能层似然的标准化管道
UBMS 将心理学数十年积累的海量行为测量工具（PHQ-9、GAD-7、Stroop、N-back、数字表型等）整合为统一的行为似然管道。数据本身已是数值形式——核心挑战不是"理解含义"（那是 UPLS 的任务），而是定义观测函数 p(yb|Ψ)，管理跨工具桥接与版本演化，并为 PSE 提供标准化的行为似然接口。
一、三层架构总览

每层回答一个独立问题，层间通过标准化 JSON Schema 通信，复用现有心理测量标准（IRT、CTT、PROMIS、COSMIN）。

三、关键设计决策

3.1 UPLS 语义对齐前置

两个量表只有在 UPLS 确认它们测量同一 M-R-D-S 组件后，才能进入 L2 进行 IRT 跨工具等价转换。这防止了将测量不同功能构念的量表强行数值桥接——例如，将 PHQ-9（抑郁）与 GAD-7（焦虑）的分数强行等价转换，在功能层上毫无意义。

3.2 复用现有心理测量标准

CCT

经典测量理论

信度指标（Cronbach's α、重测信度）作为 BHI 注册库的一等公民。

IRT

项目反应理论

跨工具桥接的数学基础，实现不同量表分数的等价转换。

COSMIN

效度评估框架

内容效度、结构效度、反应性——纳入 BHI 元数据。

3.3 个体化校准的层级收缩

L2 的贝叶斯个体化校准采用层级先验：个体的响应风格参数被建模为来自群体分布的随机效应，在观测数据不足时自动收缩至群体先验，防止过拟合。这使 UBMS 能在 N-of-1 和小样本场景中工作。

四、观测函数与 BHI 标识符

BHI · 行为健康标识符

作用： 为每个标准化行为测量工具分配唯一标识符

示例：

BHI-PHQ9 = {
  "construct": "M4-DSP-B↓ + M1-RES-D↓",
  "IRT_model": "graded_response",
  "reliability": 0.84,
  "M-R-D-S_anchor": ["M4", "M1-RES-D"]
}

观测函数

形式： yb = h(Ψ, c, θi) + ε

参数：

  • Ψ — 功能层状态向量
  • c — 工具常量（IRT 参数）
  • θi — 个体参数（反应风格、设备差异）
  • ε — 测量误差项

六、基础设施间接口

UBMS → UPLS

依赖关系： 数值桥接前必须经 UPLS 语义对齐

两个量表只有在 UPLS 确认测量同一 M-R-D-S 组件后才能 IRT 桥接

UBMS → UNIS

对等关系： 两条独立的输入通道

行为通道与神经通道各自提供独立似然，差异在 PSE 融合层解析

UBMS → PSE

输出关系： 通过 L3 行为似然 API

p(yb|Ψ) 直接注入 PSE 贝叶斯融合器，与语义、神经似然联合估计状态后验
