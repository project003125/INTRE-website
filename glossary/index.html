核心架构术语速查

核心架构术语
N-S-R | Need-Structure-Realization — 元公理：需要→结构→实现的循环
M-R-P | Module-Ring-Playbook — 通用架构：模块-调度环-剧本
M-R-D-S | Module-Resource-Dispatch-Structure — 状态语法：模块-资源-调度-结构
M1–M6 — 六模块：M1具身 / M2认知 / M3意志 / M4情绪 / M5言语 / M6行为
42维状态向量 — Ψ(t)：6模块 × 3维度(R/D/S) + 6交互项 = 42维
RES-D | Dispatchable Resource — 调度资源：即时可用的心理能量
RES-B | Basal Resource — 基底资源：长期心理资本
DSP-F | Foreground Dispatch — 前台调度：需意识参与的目标导向进程
DSP-B | Background Dispatch — 后台调度：无需意识努力的自动进程
STR-R | Rule Structure — 规则结构：系统的规则库
STR-C | Connection Structure — 连接结构：系统的权重网络
STR-S | Sequence Structure — 序列结构：系统的时间序列协议
三层架构 — 现象层 / 功能层 / 神经层，功能层为工程核心
三通道融合 — 行为似然·语义似然·神经似然 → 贝叶斯乘积融合 p(y|Ψ)=p(y_b)·p(y_s)·p(y_n)
S-P | Source-Pattern — 外部架构：环境源流与影响模式
S层六层 — 微系统 / 中系统 / 外系统 / 宏系统 / 时序系统 / 数字系统
P+ / P– — 资源增强型 / 资源消耗型环境模式
操作语法 (G-M-A-R)
GATE — 门控操作：启动/抑制/切换进程，目标 DSP
GATE+ — 启动/增强进程
GATE– — 抑制/减弱进程
GATE-redirect — 切换进程通道
MODULATE — 调制操作：调节资源水平，目标 RES-D
MODULATE+ — 补充资源
MODULATE– — 消耗资源
ALIGN — 校准操作：修正规则结构，目标 STR-R
RELINK — 重联操作：重组连接网络，目标 STR-C
心理系统工程 (PSE)
PSE | Psychological System Engine — 心理系统工程引擎
MCT | Mental Compilation Theory — 心智编译理论：观测 → 状态向量
MTT | Mental Topology Theory — 心智拓扑理论：网络结构与级联
MKT | Mental Kinetics Theory — 心智动力学理论：资源演化方程
MDT | Mental Dispatch Theory — 心智调度理论：干预优化决策
MPT | Mental Plasticity Theory — 心智可塑性理论：结构改变机制
MET | Mental Ecology Theory — 心智生态理论：环境–系统耦合
心智六论 — PSE 的六个理论模块：MCT + MTT + MKT + MDT + MPT + MET
VTS-P | Variable-to-Structure Protocol — 变量到结构协议
ETS-P | Environment-to-Structure Protocol — 环境到结构协议
SID-P | System Integrated Diagnosis Protocol — 系统集成诊断协议
MMD-P | Multi-Dimensional Meta-Dispatch Protocol — 多维元调度协议
STO-P | Structure-to-Operation Protocol — 结构到操作协议
PSE-I/M/P/C — PSE 四阶段：I 理论注入 → M 蒙特卡罗仿真 → P 先验提取 → C 真实世界校准
知识基础设施 (UPLS)
UPLS | Unified Psychological Language System — 统一心理语言系统：形式化元语言基础设施
PCUI | Psychological Concept Unique Identifier — 心理概念唯一标识符
PSA | Psychological Semantic Architecture — 心理语义架构：概念间关系定义
PSL | Psychological Specialist Lexicon — 心理专家词典：自然语言 → PCUI 映射
神经接口系统 (UNIS)
UNIS | Unified Neural Interface System — 神经通道（EEG/fNIRS/ECoG）→ 功能层似然，六层递进架构
NAP | L1 Neural Acquisition Protocol — 神经采集规范：规定 BIDS 格式、同步等级、元数据标准
SCL | L2 Signal Calibration Layer — 信号校准层：去伪影、滤波、时间同步
NFD | L3 Neural Feature Definition — 神经特征定义：分配 NFI 并提供参考实现
NFM | L4 Neuro-Functional Mapping — 神经-功能映射：NFI → p(y_n|Ψ_t) 预测模型
NPL | L5 Neural Prior Library — 神经先验库：生理硬约束 + 统计规律约束
RTE | L6 Real-time Engine — 实时引擎：含推断(RTE-Infer)与校准(RTE-Cal)双模块
NFI | Neural Feature Identifier — 神经特征标识符，如 NFI.EEG.v1.002
BIDS | Brain Imaging Data Structure — 神经数据组织标准，UNIS L1 采用
LSL | Lab Streaming Layer — 实时数据流同步协议，UNIS L6 采用
ONNX | Open Neural Network Exchange — 模型跨框架交换格式，UNIS L4 采用
神经似然 — UNIS 标准化输出 p(y_n|Ψ_t)，参与 PSE 三通道贝叶斯融合
仲裁合约 — UNIS ↔ PSE 双向通信接口：/func_likelihood（前向）+ /calibration_feedback（反向）
通道分歧 — 行为/语义/神经三通道对同子单元估计出现系统性分歧时的诊断信号
行为测量系统 (UBMS)
UBMS | Unified Behavioral Measurement System — 统一行为测量系统：FLP 行为通道基础设施，将结构化行为观测转化为行为似然
BHI 标识符 | Behavioral Health Indicator — 行为健康指标标识符，格式 BHI.[DOMAIN].v[VER].[SEQ]，如 BHI.DEPRESSION.v2.003
BHI 条目 — UBMS L1 注册库中 BHI 标识符对应的完整记录（含观测函数、信效度、桥接参数等）
BHI 注册库 — UBMS L1 中所有 BHI 条目的集合
UBMS L1 | Behavioral Measurement Registry — 行为测量注册库：存储观测函数、信效度矩阵、桥接参数、数字表型协议
UBMS L2 | Calibration & Linking Engine — 校准与桥接引擎：IRT 跨工具等价转换 + 贝叶斯个体化校准 + 漂移检测
BLA | Behavioral Likelihood API — 行为似然 API（UBMS L3）：JSON-RPC 端点 /behavioral_likelihood 和 /calibration_feedback
UBMS 观测函数 | h_b(x) — 行为通道观测函数：将测量分数 x 映射到功能状态子维度似然（方向：x→y_b）
行为似然 — BLA 标准化输出 p(y_b|Ψ_t)，必须经 UBMS L3 端点获取，参与 PSE 三通道融合
UPLS 语义对齐前置 — UBMS 核心约束：两个 BHI 条目须经 UPLS 确认同一 M-R-D-S 组件后方可进行 IRT 桥接
工具桥接 | Instrument Linking — 不同 BHI 条目对同一 M-R-D-S 组件的分数等价转换，由 UBMS L2 执行
数字表型协议 | Digital Phenotype Protocol — UBMS L1 可选字段：从原始传感器数据到数字表型特征的完整处理链定义
IRT | Item Response Theory — 项目反应理论：UBMS L2 跨工具等价转换的方法论基础
CTT | Classical Test Theory — 经典测量理论：UBMS L1 信度指标的方法论基础
PSE-M 内嵌子层 — UBMS 前身：行为测量作为 PSE-M 蒙特卡罗仿真阶段内部子模块的历史形态
FLP 学科体系
FLP | Functional Layer Psychology — 功能层心理学：INTRE 框架的学科统称，含五大支柱学科
PFIM | Psychofunctional Input Metrology — 功能层输入测量学：多通道 p(Ψ|y) 估计
PFD | Psychofunctional Dynamics — 功能层动力学：状态演化规律与级联分析
PFI | Psychofunctional Intervention — 功能层干预学：靶点明确的干预设计与优化
PFM | Psychofunctional Metascience — 功能层元科学：知识质量保障与证伪制度化
PDE | Psychofunctional Developmental Ecology — 功能层发展生态学：生命周期与跨文化维度
DMD | Default Mode Dispatch Therapy — 从 GMAR 语法空位生成的结构化心理干预
CI+AI — 协同建构方法论：人类晶核 + AI 展开
伦理引擎 (REVER)
REVER — 关系型 AI 伦理引擎：结构化伦理门控
Dₛ — 依赖分数：量化用户依恋程度
四道门控 — 输入 → 诊断 → 依赖 → 输出，四重伦理检查
证据等级标签 (IET v3.0)
D — 定义类：术语如何被规定/界定
L — 推导类：从前提可推出什么
F — 描述类：事实是什么
X — 解释类：为什么会发生
P — 预测类：未来会发生什么
N — 规范类：应该怎样
E — 证据类：有什么证据支持
R — 关系类：事物之间有什么关系
示例标签 — D1a(规定性定义) / L1a-p(必然推演) / P2i-m(假说,间接可检验) / E1a+(RCT强证据)
