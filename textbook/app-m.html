附录M：数学速成——函数、微分、概率与滤波

目标读者：具备中国高考中等数学水平、正在学习或从事心理学、认知科学、临床医学等专业的学生或研究者。本附录帮助你快速掌握阅读第8、19、20、28、29章所必需的数学概念。

学习路线：建议按“→”顺序阅读，每节约15-30分钟。可根据需要跳跃。

函数与导数 → 微分方程 → 线性代数基础 → 概率与贝叶斯 → 状态估计思想 → 粒子滤波 → 非线性动力学

M.1 函数与导数（高中回顾）

为什么需要：第8章的状态向量 Ψ(t)\Psi(t)Ψ(t) 是随时间变化的函数；导数描述“变化快慢”。

概念 通俗解释 数学形式 心理学例子
函数 一个量随另一个量变化的规则 y=f(x)y = f(x)y=f(x) 意志资源 RRR 随时间 ttt 变化：R(t)R(t)R(t)
导数 函数在某点的瞬时变化率 dydt\frac{dy}{dt}dtdy​ 或 y′y'y′ 意志资源的消耗速率 dR/dtdR/dtdR/dt
指数衰减 按当前值的固定比例减少 dRdt=−αR\frac{dR}{dt} = -\alpha RdtdR​=−αR 资源消耗速率与当前资源量成正比

关键式子：若 dRdt=−αR\frac{dR}{dt} = -\alpha RdtdR​=−αR，则 R(t)=R0e−αtR(t) = R_0 e^{-\alpha t}R(t)=R0​e−αt（指数衰减）。

📌 你不需要会推导，只需知道：指数衰减意味着“越剩越少，减少得越慢”。

M.2 微分方程基础

为什么需要：第8、19章用微分方程描述心理状态如何随时间演化。

概念 解释 第19章对应
常微分方程（ODE） 包含未知函数及其导数的方程 dRdt=供给−αR−消耗\frac{dR}{dt} = \text{供给} - \alpha R - \text{消耗}dtdR​=供给−αR−消耗
解 满足方程的函数 R(t)R(t)R(t) 资源随时间变化的轨迹
初始条件 t=0t=0t=0 时的状态值 R(0)R(0)R(0) 初始评估时的资源水平
参数 方程中不变的常数（如 α\alphaα） 消耗速率、恢复速率

核心思想：给定初始状态和演化规则，计算机可以通过数值积分（如欧拉方法）近似算出未来轨迹——你不必手工解方程，只需理解输入输出。

欧拉方法直观：R(t+Δt)≈R(t)+dRdt×ΔtR(t+\Delta t) \approx R(t) + \frac{dR}{dt} \times \Delta tR(t+Δt)≈R(t)+dtdR​×Δt，一步接一步，像动画片逐帧播放。

M.3 线性代数基础（第8章必备）

为什么需要：第8章的状态向量 Ψ(t)\Psi(t)Ψ(t) 是42维向量，涉及向量运算、正交性、欧氏距离等概念。

概念 通俗解释 数学形式 心理学例子
向量 一组有序的数字列表 v⃗=(v1,v2,…,vn)\vec{v} = (v_1, v_2, \dots, v_n)v=(v1​,v2​,…,vn​) 42维状态向量 Ψ\PsiΨ
向量的模长（范数） 向量的长度 ∥v⃗∥=v12+v22+⋯\|\vec{v}\| = \sqrt{v_1^2 + v_2^2 + \cdots}∥v∥=v12​+v22​+⋯​ 状态向量的“总偏离”程度
向量点积（内积） 两向量对应分量乘积之和 u⃗⋅v⃗=∑uivi\vec{u} \cdot \vec{v} = \sum u_i v_iu⋅v=∑ui​vi​ 用于计算相似度
欧氏距离 两向量之间的直线距离 d=∑(ui−vi)2d = \sqrt{\sum (u_i - v_i)^2}d=∑(ui​−vi​)2​ 两个心理状态的相似性
正交 两向量点积为零，互相垂直 u⃗⋅v⃗=0\vec{u} \cdot \vec{v} = 0u⋅v=0 R、D、S维度功能独立
矩阵 二维数字表格 Am×nA_{m \times n}Am×n​ 依赖矩阵（42×42）
转置 行列互换 ATA^TAT 用于计算协方差

📌 你不需要手算高维向量，只需理解：状态向量是心理系统在42个坐标轴上的读数；两个状态之间的距离越短，越相似。

M.4 概率与贝叶斯定理

为什么需要：第20章用贝叶斯校准融合观测数据，更新对状态的不确定性。

概念 通俗解释 符号
概率 事件发生的可能性（0到1） P(A)P(A)P(A)
条件概率 在B发生下A的概率 P(A∣B)P(A \mid B)P(A∣B)
贝叶斯定理 已知 P(B∣A)P(B \mid A)P(B∣A) 求 P(A∣B)P(A \mid B)P(A∣B) P(A∣B)=P(B∣A)P(A)P(B)P(A \mid B) = \frac{P(B \mid A)P(A)}{P(B)}P(A∣B)=P(B)P(B∣A)P(A)​

心理学解释： - 先验 P(A)P(A)P(A)：观测数据之前的初始信念（如生态位先验） - 似然 P(B∣A)P(B \mid A)P(B∣A)：在给定状态A下观测到数据B的可能性 - 后验 P(A∣B)P(A \mid B)P(A∣B)：观测数据后更新的信念

例子：怀疑意志资源很低（A）。做任务表现差（B）。先验30%，似然90%，后验≈\approx≈68%。这就是校准。

M.5 状态估计的思想

为什么需要：第20章的粒子滤波是卡尔曼滤波的推广。先理解核心思想。

概念 解释
状态 你关心的隐藏量（如真实意志资源水平）
观测 你能测到的量（如量表分数），但含噪声
状态估计 利用历史观测和动力学模型，猜出当前状态的最佳值

卡尔曼滤波的四个步骤（只需记逻辑）： 1. 预测：用动力学方程推测下一时刻状态 2. 观测：获得带噪声的读数 3. 更新：结合预测和观测，按不确定性加权平均 4. 重复

📌 类比：开车进隧道前知道速度（预测），隧道内推算位置（预测），出隧道后GPS恢复（观测），结合修正（更新）。

M.6 粒子滤波概念

为什么需要：当系统是非线性、非高斯时（心理系统正是如此），卡尔曼滤波失效。粒子滤波是通用替代方法。

概念 通俗解释
粒子 一个“假设”——某个可能的状态值
粒子群 成千上万个假设，覆盖各种可能性
权重 每个粒子与观测数据的符合程度
重采样 删掉低权重粒子，复制高权重粒子

粒子滤波循环：

初始粒子群（均匀散布）
    ↓
预测：根据动力学方程移动每个粒子
    ↓
观测：计算每个粒子的权重（与数据符合度）
    ↓
重采样：高权重粒子繁殖，低权重淘汰
    ↓
（循环）

最终输出：粒子群聚集区 = 状态最佳估计；分散程度 = 不确定性。

📌 类比：用一群侦探分头追踪嫌疑人，每步用新线索淘汰不靠谱侦探，让靠谱侦探繁殖。

M.7 集合预报与CRPS（第20章）

为什么需要：PSE输出概率预测，需要评估预测质量。

概念 解释
集合预报 用多个粒子（成员）的轨迹集合表示预测分布 类似气象学中的“台风路径集合”
中位数轨迹 集合中排在中间的那条轨迹 最可能的估计
预测区间 2.5%和97.5%百分位数之间的区域 95%置信区间
CRPS（连续排序概率得分） 评估整个预测分布与真实观测的吻合度 越小越好，完美预测=0

📌 CRPS惩罚过宽（太保守）和过窄（太自信）的预测区间。

M.8 非线性动力学基本概念（第29章）

为什么需要：第29章讨论混沌、吸引子、非线性系统。

概念 通俗解释
非线性系统 输出不与输入成正比，叠加原理不成立 心理系统常见（如阈值效应）
相空间 状态向量 Ψ\PsiΨ 所在的虚拟空间 42维空间，每个点代表一种心理状态
吸引子 系统长期趋向的“偏好区域” 健康状态是一种吸引子，抑郁症是另一种
混沌 确定性但不可预测：初始微小差异导致巨大不同 “蝴蝶效应”
临界慢化 系统靠近临界点（相变前）的恢复速度变慢 心理崩溃前兆

📌 这些概念只需定性理解，帮助读懂第29章的研究展望。

第28章涉及一些更专业的概念，对于理解PSE原理不是必须的。下表供好奇读者参考：

概念 简短说明 出现在第28章
